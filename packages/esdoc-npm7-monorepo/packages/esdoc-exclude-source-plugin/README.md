# ESDoc Exclude Source Plugin
## Install
```bash
npm install https://github.com/EnterTheNameHere/esdoc-exclude-source-plugin.git
```

## Config
```json
{
  "source": "./src",
  "destination": "./doc",
  "plugins": [
    {"name": "esdoc-exclude-source-plugin"}
  ]
}
```
