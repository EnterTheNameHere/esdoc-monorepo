# ESDoc External WebAPI Plugin
## Install
```bash
npm install https://github.com/EnterTheNameHere/esdoc-external-webapi-plugin.git
```

## Config
```json
{
  "source": "./src",
  "destination": "./doc",
  "plugins": [
    {"name": "esdoc-external-webapi-plugin", "option": {"enable": true}}
  ]
}
```

`enable` is default `true`.
